- Implement auth provider (Clerk first)
- Add PDF/text ingestion API and storage adapter
- Add Chroma indexing and metadata filters

- Build grade ingestion workflows
- Compute weak-topic and trend analytics
- Generate personalized recommendations and study plans

- Launch tutor and quiz agents
- Add exam calendar and announcement services
- Dashboard integration for students/lecturers

## Phase 4
- Add predictive risk model
- Lecturer cohort analytics and intervention workflow
- Observability and evaluation loops for agent quality
